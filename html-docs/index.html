02 System Modules

🔐
Identity & Access
JWT auth, refresh token rotation, RBAC (5 roles), account lockout, rate limiting
👤
Patient Management
Registration, MRN generation, demographic & insurance management, paginated search
📁
Case Management
Full case lifecycle (Draft → Closed/Escalated), status machine, tags, case history
📝
Case Notes
Append-only clinical notes, 24h edit window, immutable after lock
✅
Task Management
Case-scoped tasks, assignee, due dates, overdue detection, status transitions
📄
Document Management
Azure Blob upload (≤20 MB), 1-hour SAS download URLs, soft delete, categories
📅
Appointments
Scheduling, provider conflict detection, 4 appointment types, 5 status transitions
👥
Care Team
Per-case team composition, lead coordinator + specialists, audit-logged changes
📊
Dashboard & Reports
3 role-specific dashboards, Chart.js widgets, CSV export for all report types
🔍
Audit Log
Immutable HIPAA-aligned event log — entity, action, old/new JSON, user, IP, timestamp
🔔
Notifications
In-app bell, unread badge, trigger engine (case assign, task overdue, escalation)
⚙️
Administration
User management, role assignment, audit log viewer (admin only)
NEW
⚕️
FHIR/HL7 Integration
FHIR R4 REST facade, Patient/$everything Bundle, import/export, HL7 v2 ADT+ORU receiver (stretch)
